### Step 1: Show Client the Preview
```bash

start blue-schemes-preview.html
```

### Step 2: Get Feedback
- Have client click through each blue scheme

- Note any specific elements they like/dislike

### Step 3: Apply Chosen Scheme

**Option A - Quick Test (Recommended for testing):**
Add to all HTML pages after `<link rel="stylesheet" href="styles.css">`:
```html
<link rel="stylesheet" href="color-schemes/blue-X-name.css">
```

**Option B - Permanent Integration:**
1. Open the chosen scheme CSS file
2. Copy the `:root` variables (lines 8-21)
3. Replace the `:root` section in `styles.css` (lines 8-21)
4. Update hardcoded gradients in `styles.css`:
   - Line 135 (Hero gradient)
   - Line 991 (About hero gradient)
   - Line 1275 (CTA gradient)
   - Line 20 (Shadow color)
